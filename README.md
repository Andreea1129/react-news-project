# Description
This project is a news web application built with React. 
The application allows users to browse multiple news articles from different categories. 
The home page displays a selection of news from each category, along with links to access the dedicated category pages.

Each category has its own dedicated page. 
Users can save their favorite articles by clicking the “Adaugă la favorite” button available for each individual news item. 
When an article is added to favorites, a confirmation message is displayed.
All saved articles can be accessed in the Favorites section, where users can also remove articles they no longer wish to keep.

# Features
- View latest news articles
- Read detailed news content
- Add and remove articles from favorites
- Persist favorites using localStorage
- Responsive design using Bootstrap

# Technologies Used
- React
- React Router
- Context API
- JavaScript (ES6+)
- Bootstrap
- The Guardian API
- CSS
- HTML
